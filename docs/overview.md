# 项目概述与技术说明书

## 一、项目定位

面向工业重载、应急救援场景的开源液压外骨骼方案，采用「车载动力单元 + 外骨骼执行端」分体设计，实现零力人机交互、百公斤级负载承载、全地形稳定行走。

本项目为纯液压驱动的重载外骨骼动力机甲技术方案，核心设计围绕四大原则展开：
- 双泵 100% 物理隔离
- 上下半身分时工作
- 人机零力交互
- 全标品元件复用

所有核心液压、结构、电控元件均采用市面通用量产标品，最大化降低复刻与维护成本。

## 二、核心技术参数

| 约束类别 | 核心参数指标 |
| :--- | :--- |
| **承载性能** | 总负载 750 kg（外骨骼自重 150 + 人体 100 + 双臂负载 500）；单臂额定 300 kg，双臂额定 500 kg |
| **液压系统** | 双独立回路，额定 16 MPa，峰值 21 MPa，极限 25 MPa；单泵额定流量 9.6 L/min；下半身 0.6 L 蓄能器，上半身 1.6 L 蓄能器 |
| **关节配置** | 共 18 DOF：下肢 12 路（髋 3×2 + 膝 1×2 + 踝 2×2），上肢 6 路（三轴机械臂×2） |
| **行走性能** | 平地 0.5 m/s；连续负重 ≥10 min；步态角度跟踪误差 ≤±0.5° |
| **关节角度** | 膝屈曲最大 90°；髋屈伸 ±60°；踝俯仰 ±45°、内翻外翻 ±45°；上肢举升 ±90° |
| **结构材料** | 主承载框架：7075-T6 航空铝合金；高强度铰点/销轴：40Cr 调质钢 |

| **控制架构** | 主控制器 + 协处理器双核心，下半身 1 kHz，上半身 100 Hz |

## 三、核心设计原则

### 3.1 重载闭环承载原则
- 负载零人体承担：单臂 300 kg / 双臂 500 kg 重载负载，100% 由外骨骼刚性框架闭环传递至足底
- 上下半身分时互锁：硬件与软件双层杜绝动力冲突与重心失控
- 长效保压能力：支撑相静态保压 30 分钟位置漂移 ≤1 mm

### 3.2 人机零力交互原则
- 无刚性绑定：仅臀部坐垫、足部平台两处无受力接触，三点式柔性安全带限位
- 被动随动行走：下肢带动人体腿部被动随动，人体无需发力
- 姿态恒定约束：全程维持上半身铅锤正直，脚底与支撑面完全平行
- 普适性适配：覆盖 160/175/190 cm 三身高人群

### 3.3 双泵独立液压原则
- 物理完全隔离：1 号泵专属下半身 12 路，2 号泵专属上半身 6 路，无油路串扰
- 全标品元件：18 个自由度液压缸 100% 国内量产通用标品
- 差异化阀件选型：下半身高频响小通径比例阀，上半身大通径重载比例阀

### 3.4 分体式接口原则
- 动力单元集成于配套小车，≤5 m 高压铠装软管连接
- 背部/底部标准化快拆对接框架，10 秒快速拆装锁止

## 四、系统整体架构

「车载动力层 + 分体阀块控制层 + 分级执行层 + 全链路安全层」四层独立双回路架构。

### 4.1 动力层
- 双电机双出轴，各驱动一台 5 mL/r 高压低脉动齿轮泵
- 独立微型蓄能器就近安装于对应阀块，压力波动 ≤±1 MPa
- 动力单元集成于动力小车，通过高压铠装软管分体传输

### 4.2 控制层
- 主控制器（STM32F407）+ 协处理器（STM32F103）双核心
- 18 路关节分优先级独立闭环控制
- 双闭环逻辑：内环阀芯位置闭环 + 外环关节角度闭环
- 五大核心模块：ZMP 平衡解算 / 三身高自适应 / 7 场景步态 / 分时互锁 / 多模式切换

### 4.3 执行层
按负载分级配置 3 款国标通用标品液压缸：
- **主承载关节**（髋膝、上肢重载）：50/36 缸径等速双向液压缸
- **辅助关节**（髋肩旋转/侧摆）：32/18 双向液压缸
- **微调关节**（踝关节）：25/14 双向液压缸

### 4.4 安全层
- 基础安全：双系统独立全链路压力保护、多点位急停、超程限位、强弱电隔离
- 专项安全：躯干倾角超限保护、关节角度超程锁止、脚底失贴预警、管路防爆、快速脱扣

## 五、工作模式

| 模式 | 外骨骼状态 | 应用场景 |
| :--- | :--- | :--- |
| **小车载人** | 全关节锁止，双泵卸荷，整体随小车行驶 | 高速转场、长距离机动 |
| **行走+跟随** | 1 号泵驱动下肢 12 路，小车同步跟随（速度误差 ≤0.02 m/s） | 复杂地形通行、废墟搜救 |
| **重载作业** | 双泵全功率输出，严格上下半身分时工作 | 重物搬运、破拆救援 |

## 六、性能校核基准

### 6.1 极限负载压力校核（动载系数 2.5）
- 膝关节：最大工作压力 15.18 MPa < 16 MPa 额定 ✓
- 髋关节主屈伸：工作压力 ≤2 MPa，冗余充足 ✓
- 上肢单臂 300 kg 举升：最大工作压力 14.5 MPa < 16 MPa 额定 ✓

### 6.2 流量与连续工作校核
- 常规步频 48 步/min（周期 1.25 s），单侧摆动腿耗油 0.1907 L
- 1 号泵 + 蓄能器峰值补偿，可实现 30 min 以上连续负重行走

### 6.3 结构强度校核
- 7075-T6 铝合金主框架：安全系数 ≥4.0 ✓
- 40Cr 调质钢销轴：抗剪强度 ≥600 MPa，疲劳寿命 ≥10 万次 ✓
- 高压铠装软管：疲劳寿命 ≥10 万次弯折，耐压冗余 ≥2 倍极限 ✓

## 七、开发迭代路线

| 阶段 | 内容 |

| **Phase 0** | 方案定型与设计校核 — 液压原理图、运动学建模、强度校核、BOM |
