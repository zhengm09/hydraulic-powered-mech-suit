# Hydraulic-Powered Mech Suit | 液压动力机甲

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](LICENSE)
[![Status](https://img.shields.io/badge/Status-Phase%200%20%E2%80%94%20Design-red)]()

面向工业重载、应急救援场景的**开源液压外骨骼动力机甲**。采用「车载动力单元 + 外骨骼执行端」分体设计，实现零力人机交互、百公斤级负载承载、全地形稳定行走。

> ⚠️ **Phase 0 — 方案定型阶段**。当前为概念设计与技术说明书，硬件尚未打样。

## 核心指标

| 参数 | 值 |
| :--- | :--- |
| **总负载** | 750 kg（自重 150 + 人体 100 + 双臂 500） |
| **液压压力** | 额定 16 MPa / 峰值 21 MPa / 极限 25 MPa |
| **自由度** | 18 DOF（下肢 12 + 上肢 6） |
| **行走速度** | 0.5 m/s（平地负重） |
| **结构材料** | 7075-T6 铝合金 + 40Cr 调质钢 |
| **控制架构** | STM32F407 + F103 双核，1 kHz / 100 Hz |

## 四大设计原则

1. **重载闭环承载** — 负载 100% 由刚性框架传递至足底，零人体承担
2. **人机零力交互** — 无刚性绑定，被动随动行走，三点式柔性安全带限位
3. **双泵独立液压** — 上下半身物理完全隔离，18 路关节全标品液压缸
4. **分体式接口** — 动力单元集成于配套小车，≤5m 高压铠装软管连接

## 三种工作模式

| 模式 | 状态 | 场景 |
| :--- | :--- | :--- |
| **小车载人** | 外骨骼待机，全关节锁止 | 高速转场、长距离机动 |
| **行走+跟随** | 1号泵驱动下肢，小车跟随 | 复杂地形、废墟搜救 |
| **重载作业** | 双泵全功率输出 | 重物搬运、破拆救援 |
